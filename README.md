# Training Planner

**Автор:** Донцов Дмитрий

## Описание

Training Planner - это графическое приложение для планирования и учета тренировок. Приложение позволяет добавлять тренировки, фильтровать их по дате и типу, а также сохранять данные в формате JSON.

## Возможности

- Добавление тренировок с указанием даты, типа и длительности
- Валидация ввода данных (формат даты, положительное число для длительности)
- Фильтрация тренировок по типу и диапазону дат
- Автоматическое сохранение данных в JSON
- Экспорт и импорт данных в JSON
- Подсчет статистики (общее количество тренировок, общая и средняя длительность)
- Удобный интерфейс с таблицей и скроллингом

## Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки (tkinter, json, datetime) - обычно уже установлены

### Запуск
```bash
python training_planner.py
Примеры использования
Пример 1: Добавление тренировки
Заполните поля:

Дата: 2024-01-15

Тип: Бег

Длительность: 45

Нажмите Добавить тренировку

Тренировка появится в таблице

Пример 2: Фильтрация
Выберите тип: Плавание

Укажите диапазон дат: от 2024-01-01 до 2024-01-31

Нажмите Применить фильтр

В таблице отобразятся только тренировки по плаванию за январь

Пример 3: Сохранение и загрузка
Автосохранение происходит в файл trainings.json

Для ручного экспорта нажмите Сохранить в JSON (создаст trainings_export.json)

Для загрузки из файла нажмите Загрузить из JSON
